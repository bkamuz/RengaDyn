# RengaDyn Documentation

This directory contains comprehensive documentation for the RengaDyn project, organized by category for easy navigation.

## Directory Structure

### `/api/`
API guides and reference documentation:
- **POLYCURVE_CONVERSION_GUIDE.md** - Complete guide for polycurve conversion
- **OPTIMIZED_POLYCURVE_GUIDE.md** - Optimization techniques for polycurve handling
- **CURVE_TYPE_AWARE_CONVERSION.md** - Type-aware curve conversion strategies
- **CLOSED_POLYCURVE_SOLUTION.md** - Solutions for closed polycurve handling
- **CLOSED_POLYCURVE_SOLUTION_SPECIFIC.md** - Specific implementations for closed polycurves
- **RENGA_COORDINATE_SYSTEMS_GUIDE.md** - Coordinate systems in Renga

### `/workflows/`
Step-by-step workflow documentation:
- **baseline-workflow.md** - Complete Baseline2DObject workflow process
- **polycurve-conversion-workflow.md** - Curve2D to PolyCurve conversion workflows

### `/troubleshooting/`
Troubleshooting guides and common issues:
- **TROUBLESHOOTING_BASELINE2D.md** - Comprehensive Baseline2DObject troubleshooting
- **COM_ERROR_E_FAIL_BASELINE.md** - ⚡ **NEW** - COM E_FAIL error when setting baselines
- **BASELINE2D_USAGE.md** - Best practices for using Baseline2D objects
- **NULL_OUTPUT_TROUBLESHOOTING.md** - Debugging null output issues
- **CURVE_CONVERSION_TROUBLESHOOTING.md** - Common curve conversion problems and solutions

### `/notes/`
Development notes and version history:
- **Заметки.txt** - Developer notes in Russian
- **Обновления.txt** - Version history and updates in Russian

## Getting Started

1. **For API usage**: Start with the `/api/` directory guides
2. **For implementation**: Follow the workflows in `/workflows/`
3. **For debugging**: Check `/troubleshooting/` for common issues
4. **For project history**: Review `/notes/` for development context
