## 🏗️ Новая архитектура проекта

### 📁 Структура папки Core/

```
src/Core/
├── BaseRengaWrapper.cs           # Базовый wrapper для всех интерфейсов Renga
├── BaseRengaStyle.cs             # Базовый класс для всех стилей
├── BaseRengaParameter.cs         # Базовый класс для параметров
├── GeometryFactory.cs            # Фабрика геометрических объектов
└── Interfaces/
    ├── IRengaIdentifiable.cs     # Интерфейс для объектов с ID
    └── IRengaNamed.cs           # Интерфейс для именованных объектов
```

### 🔧 Ключевые базовые классы:

#### 1. `BaseRengaWrapper<TInterface>`
```csharp
// Универсальный wrapper для всех COM-интерфейсов Renga
public abstract class BaseRengaWrapper<TInterface> where TInterface : class
```
**Функции:**
- Типобезопасная инициализация COM-интерфейсов
- Валидация и отладочная информация
- Устранение дублирования конструкторов

#### 2. `BaseRengaStyle<TStyleInterface>`
```csharp
// Базовый класс для всех стилей (BeamStyle, ColumnStyle, etc.)
public abstract class BaseRengaStyle<TStyleInterface> : BaseRengaWrapper<TStyleInterface>
```
**Функции:**
- Унифицированное API для всех стилей
- Автоматическая реализация Id, Name, UniqueId
- Расширенная отладочная информация

#### 3. `GeometryFactory`
```csharp
// Фабрика для создания геометрических объектов
public static class GeometryFactory
```
**Функции:**
- Универсальные generic методы создания точек/векторов
- Автоматическое извлечение координат из любых геометрических объектов
- Замена дублированного кода в Base.cs

```csharp
// Старый способ
var point = new Point3D();
```
